Blog API Health

Node.js · Express · TypeScript · MongoDB

Session-authenticated REST API for a publishing platform.

A compact backend project with users, admin-only post management, comments, likes, view counts, and MongoDB persistence.

Auth

Session login

Users sign in with bcrypt-protected passwords and receive session-based access.

Posts

Admin workflow

Admins can create, update, and delete published posts with tag support.

Social

Comments and likes

Authenticated users can comment, edit their own comments, and toggle likes.

Core Routes

  • POST /userRegister user
  • POST /user/authLog in
  • GET /postList posts
  • POST /postCreate post as admin
  • POST /post/:id/commentsCreate comment
  • PUT /post/:id/likeToggle like
